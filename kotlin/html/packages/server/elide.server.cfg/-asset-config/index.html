AssetConfig

@ConfigurationProperties(value = "elide.server.assets")
public final class AssetConfig

Configuration loaded at runtime which governs Elide's built-in asset serving tools.

Parameters

enabled

Whether the asset system is enabled.

prefix

URI prefix where static assets are served.

etags

Whether to generate, and respond to, ETag headers for assets.

preferWeakEtags

Whether to prefer weak ETags. Defaults to false.

rewriting

Whether to enable or disable rewriting (globally -- all rewriting features).

hashLinks

Whether to rewrite asset links based on their content hashes.
