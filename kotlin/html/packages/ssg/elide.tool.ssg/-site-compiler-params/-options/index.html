Options

public final class Options

Options payload for programmatic invocation of the SSG compiler.

Parameters

classpath

Extra JARs to add to the app execution classpath.

verbose

Whether to emit verbose log messages.

debug

Whether to enable debugging features.

httpMode

Whether to run against the app in HTTP mode.

ignoreCertErrors

Whether to ignore certificate errors.

crawl

Whether to crawl HTML content for additional URLs.

extraOrigins

Extra origins to consider legal for crawling.

timeout

Timeout to apply when executing app requests. Expressed in seconds.

precompress

Whether to pre-compress assets as *.gz or *.br files, or both, or none.

pretty

Whether to enable color and other pretty terminal output features.
