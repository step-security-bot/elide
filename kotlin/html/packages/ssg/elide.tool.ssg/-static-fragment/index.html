StaticFragment

public class StaticFragment

Internal payload for a single static site fragment, associated with the HTTP request that produced it.

Parameters

parent

Parent endpoint which produced this fragment, if applicable.

explicitUrl

Known explicit URL for this fragment; only available for crawled fragments.

expectedType

Expected type; only available for crawled fragments.

request

HTTP request which produced this site fragment.

endpoint

HTTP endpoint specification for this site fragment.

response

HTTP response produced for this fragment.

content

Parsed data which should be saved as file content.

discovered

Set of additionally-discovered transitive static fragment specs to resolve.
