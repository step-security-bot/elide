ServiceHealthManager

@Context()
@Singleton()
public final class ServiceHealthManager

Dedicated manager for service health signals; controls the central gRPC health checking service.

When a service is mounted via the RpcRuntime, it is registered with the health service. Settings for Elide's RPC layer govern whether health methods are exposed to callers.
