Error

public final class Error extends GrpcWebCallResponse

Response structure which carries information for a gRPC Web call which encountered a fatal error, including the ultimate status and any cause information.

Space for trailers are provided, but using error trailers is completely optional.

Parameters

status

Terminal status which should be assigned to this error state. Defaults to Status.INTERNAL.

contentType

Content type to use for this response. Defaults to GrpcWebContentType.BINARY.

cause

Cause information for this error, if known. Defaults to null.

headers

Header metadata which was captured as part of this response cycle.

trailers

Response trailers to enclose which describe this error state, if available.
