JsRuntimeConfig

public JsRuntimeConfig JsRuntimeConfig(Boolean enabled, Boolean sourceMaps, Boolean v8, Boolean wasm, JsRuntimeConfig.JsEsmConfig esm, JsRuntimeConfig.JsNpmConfig npm, Boolean typescript, EmbeddedScriptMetadata.JsScriptMetadata.JsLanguageLevel language, Locale defaultLocale, ZoneId timezone, Charset charset)

Parameters

enabled

Whether JS VM support is enabled.

sourceMaps

Whether to enable source-maps support, which enhances stack-traces, logs, and other system features with information about the original source code. Defaults to true.

v8

Run with V8 compatibility mode active. Defaults to false; users are not encouraged to activate this.

wasm

Enable WASM support and related bindings. Defaults to true; only active where supported.

esm

Settings which apply to ECMAScript Module (ESM) support.

npm

Settings which apply to NPM/node_modules support.

typescript

Enable experimental built-in runtime support for TypeScript. Defaults to false.

language

ECMA Script language level to apply within the VM; defaults to JsLanguageLevel.ES2022.

defaultLocale

Default locale to apply to the JS VM. Defaults to the system default.

timezone

Default timezone to apply to the JS VM. Defaults to the system default.

charset

Default character set to apply when exchanging raw data with the JS VM. Defaults to UTF-8. UTF-8 and UTF-32 are explicitly supported; other support may vary.
